# 品高即时消息客户端(Java)

品高即时消息客户端是[品高即时消息服务](http://dev.bingocc.com/im)的客户端，调用了即使消息服务的webapi接口。

## 安装

**Maven**

```xml
<dependency>
	<groupId>net.bingosoft.oss</groupId>
	<artifactId>sso-client</artifactId>
	<version>[1.0.0,]</version>
</dependency>
```

## 运行环境

|SDK版本|Java版本|
| ------ | -------  |
|1.x.x   |6+        |
